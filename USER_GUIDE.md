# SmartCopyBook — User Guide

A guide to using SmartCopyBook for taking and organizing class notes.

---

## Getting Started

1. Open SmartCopyBook in your browser (or as an installed app).
2. Click **"+ New note"** to create your first note.
3. Start typing — notes are saved automatically as you write.

---

## Creating Notes

### New note

- Click **"+ New note"** in the sidebar.
- Use the dropdown (▾) next to it to choose a template:
  - **Blank** — Empty note
  - **Lecture** — Title, date, and sections
  - **Lab Report** — Hypothesis, procedure, results
  - **Reading Notes** — Title, author, summary, key points

**Keyboard shortcut:** `Ctrl+N` (Windows/Linux) or `Cmd+N` (Mac) to create a new note.

### Editing a note

- Click a note in the list to open it.
- Edit in the text area.
- Changes are saved automatically.

---

## Organizing Notes

### Subjects

Each note has a subject. Default subjects:

- Math
- Science
- English
- Other

You can add custom subjects when creating or editing a note.

### Filtering

- Click a subject chip in the sidebar to show only notes for that subject.
- Click it again to clear the filter.

### Search

- Use the search box to find notes by **title** or **content**.
- Results update as you type.

---

## Formatting with Markdown

SmartCopyBook supports Markdown. Use the **Write** / **Preview** toggle to switch between editing and rendered view.

| You type | You see |
|----------|---------|
| `**bold**` | **bold** |
| `*italic*` | *italic* |
| `# Heading` | Large heading |
| `## Subheading` | Smaller heading |
| `- item` | Bullet list |
| `1. item` | Numbered list |
| `` `code` `` | `code` |
| ` ``` ` (code block) | Code block |

**Shortcuts while editing:**

- `Ctrl+B` — Bold
- `Ctrl+I` — Italic

---

## Exporting Notes

- Open a note.
- Click **Export** (or the export button).
- A `.md` file is downloaded so you can back up or share the note.

---

## Themes

- Use the **☀** / **☾** icon in the sidebar to switch between light and dark theme.
- Your choice is saved for next time.

---

## Installing as an App (PWA)

- **Desktop:** Use the install icon in the address bar or the browser menu.
- **Mobile:** Use "Add to Home Screen" in the browser menu.

Once installed, SmartCopyBook works offline.

---

## Where Are My Notes Stored?

Notes are stored in your browser’s **localStorage**. They stay on your device and are not sent to a server.

**Important:** Clearing browser data (cookies, site data) can delete your notes. Use **Export** regularly to back up important notes.
